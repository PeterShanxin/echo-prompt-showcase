# Security

## Scope

This repository is documentation-led and intentionally excludes deployable production systems, credentials, and sensitive infrastructure details.

## Reporting

If you believe you have identified a security issue related to materials in this repository, please report it privately to the Echo Prompt team.

Please do not open a public issue for potential security concerns.

Until a dedicated security mailbox is published, use a private GitHub contact route with the repository owner.

## Additional Notes

- No secrets should be committed to this repository
- Screenshots and examples should remain sanitized
- Infrastructure details, access patterns, and internal controls should stay private
